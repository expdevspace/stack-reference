# @kodexalabs/stack-reference

Production-ready. Powered by KodexaLabs design tokens (`@kodexalabs/design-tokens`) and CI/CD enforcement.

- Design Language: ASD tokens (see `packages/asd-tokens/`)
- CI: GitHub Workflows (`.github/workflows/ci.yml`)
- Animations: GSAP (Quantum 3D ready)
- Style: No raw values, tokens only.

## Getting Started

npm install
npm start

---

All contributions must comply with ASD and design-token standards.
